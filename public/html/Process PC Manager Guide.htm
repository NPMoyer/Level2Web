Process PC Manager User Guide

Main Screen



This screen shows up to 20 PCs. Each box contains the PC host name and nickname at the top.
The status will be either online or offline with a green or red box, respectively.
The IP address is the standard IPv4 address ranging from 0.0.0.0 to 255.255.255.255.
The reply time is the average reply in milliseconds.

If there are less than 20 PCs being pinged, then not all the boxes will be filled.
The boxes are filled from left to right and top to bottom automatically.

Different configurations can be selected from the drop-down list. Press Open to open the list and populate the screen or Remove to delete the configuration for the list.
The pinging interval and timeout time can be set using the boxes and Apply button.
Manage will open the management window and Configure will open the configuration window.

Configure



This screen allows users to configure the PCs that are listed on the main screen.
