Version 217 - PCA 2444
Released 30-Apr-2018 by NPM

New features
    FM Setup Data:
  • New L2 status area
  • Green light: L2 roll data is good
  • No light: Error in L2 roll data, FM ready is disabled on L1
    When roll data is good, the light will come back and the message will clear
  • Text box will display info about roll changes when applicable
    Roll Inventory:
  • More informational pop up messages and error checking for roll data
  • "Refresh Data" button can be used to update the screen
    No need to open and close the screen anymore
    Alarms:
  • New informational alarm when a roll change/extraction is ended, data is sent to L2
  • New critical alarm when roll data cannot be verified
  • New critical alarm when L2 tries to calculate crown for a missing roll
    There will be no setup for this slab if discharged from RHF
Bug Fixes
    Roll Inventory:
  • Screen can no longer be closed during a roll change/extraction
